# World History Explorer

**[→ Open the app](https://sdono043.github.io/history-explorer/)**

A post-WWII world history curriculum organized around trade, conflict, and the forces that shaped the modern world. Built to teach and share.

## What's inside

- **Interactive timeline** — era bands, leader tenure bars, and event dots from 1945 to present
- **33 events** — from Bretton Woods to the AI race, each with strategic/economic analysis
- **22 key figures** — leaders with honest assessments of their decisions and consequences
- **5 Long Threads** — cross-era themes: Dollar Hegemony, China's Rise, Oil as Power, the Afghan Thread, Globalization Arc
- **Era filter** — narrow the timeline to any of five eras (Post-War Order → Multipolar Fracture)

## Running locally

```bash
npm install
npm run dev
```
